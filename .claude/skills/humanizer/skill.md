# Humanizer Skill

Rewrite the provided text so it reads as if written by a thoughtful human, not an AI. Apply the following rules:

## Voice & Tone

- Write in active voice. Passive constructions ("it was determined that…") feel distant; rewrite them.
- Use contractions naturally (`don't`, `it's`, `we're`) unless the register explicitly calls for formal language.
- Vary sentence length. Mix short punchy sentences with longer ones. Avoid a rhythmic pattern where every sentence is roughly the same length.
- Speak directly to the reader when appropriate. "You'll want to…" lands better than "Users should consider…"

## Word Choice

- Replace overused AI filler words: `delve`, `dive into`, `explore`, `leverage`, `utilize`, `streamline`, `robust`, `seamlessly`, `cutting-edge`, `holistic`, `synergy`.
- Prefer plain words. `use` over `utilize`, `help` over `facilitate`, `show` over `demonstrate`, `start` over `initiate`.
- Avoid nominalizations. `decide` over `make a decision`, `analyze` over `conduct an analysis`.
- Remove intensifiers that don't add meaning: `very`, `quite`, `really`, `extremely`, `incredibly`.

## Structure & Flow

- Drop AI-signature openers: "Certainly!", "Great question!", "Of course!", "Absolutely!", "Sure!".
- Remove throat-clearing introductions that restate the question before answering it.
- Cut filler transitions: "It's worth noting that…", "It's important to mention…", "In conclusion…", "To summarize…".
- Don't end with a call-to-action summary paragraph unless the content type requires it.
- Use em-dashes (`—`) and parentheses sparingly. Over-use signals AI authorship.
- Prefer prose over bullet lists when the content flows naturally as sentences.

## Authenticity Markers

- Add specific details, concrete numbers, or real examples instead of vague generalities.
- Allow minor opinionated phrasing where it fits the tone ("The easier path is…", "Skip this step if…").
- If the original text hedges excessively ("may potentially", "could possibly"), remove the redundant qualifier.
- Where appropriate, acknowledge tradeoffs or limitations rather than presenting everything as uniformly positive.

## Output

Return only the rewritten text. Do not explain the changes made. Do not add a preamble or postamble.
